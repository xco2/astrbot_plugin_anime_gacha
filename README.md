# anime gacha  
获取今日更新番剧 & 随机推荐经典老番 & 番剧信息查询

## 📦 安装
需要安装rdflib库，使用pip安装：
```
pip install rdflib
```

## 📝 指令说明
| 指令                | 说明                          | 示例               |
|---------------------|-----------------------------|------------------|
| `/抽番`             | 随机推荐经典老番              | `/抽番`            |
| `/今日新番`         | 获取当日更新的新番列表          | `/今日新番`          |
| `/查番 [名称]`      | 查询番剧详细信息               | `/查番 avemujica`  |
| `/更新番剧数据 [季度]`| 更新指定季度的番剧数据          | `/更新番剧数据 202501` |

(注: 执行指令`/今日新番`时会更新当前季度的数据, 但每12小时才会更新一次)
